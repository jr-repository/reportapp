Lakukan pembaruan pada aplikasi dengan ketentuan berikut, tanpa mengubah flow maupun fungsionalitas yang sudah ada saat ini:

1. Pembaruan Palet Warna
- Ubah keseluruhan palet warna menggunakan kombinasi dari tiga gambar referensi yang dilampirkan.
- Gunakan tiga warna utama dari gambar tersebut sebagai dasar palet.
- Terapkan kombinasi warna dan gradient yang smooth menggunakan palet tersebut.
- Hanya ubah warna, jangan mengubah tampilan, layout, grid, maupun struktur UI lainnya.

2. Optimalisasi Action & Label
- Semua action/label seperti edit, view detail, action, generate PDF, dan lainnya diubah menjadi ikon kecil saja tanpa teks agar lebih hemat ruang.

3. Implementasi Logo
- Gunakan logo dari path: public/Assets/Image/logo.png
- Sesuaikan ukuran logo seperti standar logo aplikasi pada umumnya (proporsional).

4. Perbaikan Halaman Login
- Redesain halaman login agar lebih menarik dan profesional.
- Gunakan logo di posisi center.
- Tampilkan nama aplikasi dan tagline.

5. Perbaikan Generate PDF
- Ubah hasil generate PDF menjadi lebih rapih, terstruktur, dan profesional (skala enterprise).
- Jangan hanya seperti print halaman.
- Hilangkan tampilan seperti container UI.
- Tambahkan kop/header laporan.
- Gunakan layout laporan yang formal dan profesional.

6. Animasi Halaman
- Tambahkan animasi menggunakan AOS (Animate On Scroll).
- Terapkan secara smooth saat membuka halaman agar tidak terasa flat.

7. Perbaikan Input Field di Mobile
- Pastikan saat membuka keyboard di HP, field/input tidak tertutup keyboard.
- Field yang aktif harus tetap terlihat (auto adjust/scroll).

8. Fitur PWA (WAJIB)
- Implementasikan fitur Progressive Web App (PWA).
- Saat dibuka di browser HP, tampilkan notifikasi untuk install aplikasi.
- Gunakan logo yang telah disediakan.
- Tambahkan splash screen/opening screen:
  - tampilkan logo, nama aplikasi, dan tagline
  - gunakan animasi AOS
  - tampilkan saat pertama kali atau setiap membuka aplikasi seperti aplikasi native.

9. Update Nama Aplikasi
- Gunakan nama dan tagline berikut di seluruh aplikasi:
  TRACE — Tracking Report & Activity Control Engine

10. Ketentuan Utama (PENTING)
- Semua perubahan harus dilakukan tanpa merusak flow aplikasi dan fungsionalitas yang sudah ada.
- Fokus hanya pada visual, UX enhancement, dan UI improvement.

Silakan eksekusi seluruh instruksi di atas secara menyeluruh dan konsisten.
